# Changelog for Soft Robot HQ

All notable changes to this project will be documented in this file. Follows [Semantic Versioning](https://semver.org/).

## [v1.1.0] - 2025-12-17

### Added

* Modular Python 3D simulation using Ursina engine.
* CEO character (player) movement with WASD controls.
* Soft robots with squash/stretch animation performing basic tasks.
* Multi-floor building: lobby, manufacturing floor, CEO office, and control room.
* Privacy system with dynamic weakness calculation over time.
* Lobby visual brightness changes according to privacy weakness.

### Changed

* Initial project structure organized into separate modules for clarity.
* Console outputs privacy weakness every frame for testing.
